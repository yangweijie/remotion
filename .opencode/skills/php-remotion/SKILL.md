## Overview

PHP Remotion 是将 JavaScript [Remotion](https://remotion.dev) 核心概念移植到 PHP 的程序化视频/动画合成库。使用纯 PHP 代码创建 GIF 动画。

**核心能力:**
- 程序化动画合成
- GIF 渲染输出（内存优化 92%+）
- 时间序列管理
- 数值/颜色插值
- 物理弹簧动画
- 丰富的缓动函数
- Grafika 图像处理支持

## Features

- **Composition** - 视频合成配置（帧率、尺寸、时长）
- **Sequence** - 时间序列，支持图层组合
- **Interpolate** - 数值与颜色插值
- **Spring** - 物理弹簧动画
- **Easing** - 丰富的缓动函数（贝塞尔曲线、弹性、回弹等）
- **Layers** - 颜色层、渐变层、图片层、文字层

## Usage

### 基本用法

```php
<?php
require_once __DIR__ . '/vendor/autoload.php';

use Yangweijie\Remotion\Remotion;
use Yangweijie\Remotion\Core\RenderContext;

// 创建合成
$comp = Remotion::composition(
    id: 'my-animation',
    renderer: function (RenderContext $ctx): \GdImage {
        $frame = $ctx->getCurrentFrame();
        $config = $ctx->getVideoConfig();
        
        // 创建画布
        $canvas = Remotion::createCanvas($config->width, $config->height, [20, 20, 50]);
        
        // 动画逻辑
        $opacity = Remotion::interpolate($frame, [0, 30], [0.0, 1.0], [
            'extrapolateLeft' => 'clamp',
            'extrapolateRight' => 'clamp',
        ]);
        
        // 绘制文字
        $textLayer = Remotion::textLayer('Hello, PHP!', [
            'fontSize' => 5,
            'r' => 255, 'g' => 255, 'b' => 255,
            'align' => 'center',
        ]);
        $textLayer->setOpacity($opacity);
        $textLayer->drawOn($canvas, 0, (int) ($config->height / 2));
        
        return $canvas;
    },
    durationInFrames: 60,
    fps: 30,
    width: 640,
    height: 360,
);

// 渲染为 GIF
$renderer = Remotion::render($comp);

// 标准渲染
$renderer->renderToGif('output.gif');

// 流式编码 - 内存节省 92%（推荐长视频）
$renderer->renderToGifStreaming('output.gif');

// 优化压缩 - 最小文件体积
$renderer->renderToGifOptimized('output.gif', [], [
    'lossy' => 50,     // 有损压缩级别 (0-200)
    'colors' => 64,    // 颜色数 (2-256)
    'optimize' => 3,   // 最高优化级别
]);
```

## GIF 渲染方式对比

| 方式 | 系统内存峰值 | 文件大小 | 适用场景 |
|------|-------------|---------|---------|
| `renderToGif()` | 741 MB | 13.7 MB | 短视频 <60帧 |
| `renderToGifStreaming()` | **57.8 MB** | 1.2 MB | 内存敏感场景 |
| `renderToGifOptimized()` | **49 MB** | 9.3 MB | 最小文件体积 |

## Troubleshooting

### 内存不足

```bash
php -d memory_limit=512M example.php fade-text
```

### 中文字符显示问题

需要使用支持中文的 TTF 字体，修改 TextLayer 使用 `imagettftext` 替代 `imagestring`。

### GIF 文件过大 / 内存不足

```php
// 使用流式编码（内存节省 92%）
$renderer->renderToGifStreaming('output.gif');

// 或使用优化压缩
$renderer->renderToGifOptimized('output.gif', [], [
    'lossy' => 50,
    'colors' => 64,
]);
```

**内存对比（300帧 1080p）:**
- GD 原生: 741 MB
- 流式编码: 57.8 MB
- 优化方式: 49 MB
