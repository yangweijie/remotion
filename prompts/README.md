# 视频帧分析提示词指南

## 提示词选择策略

| 场景 | 推荐提示词 | 预期输出 | 适用模型 |
|------|-----------|---------|---------|
| 详细技术实现 | `video-analysis-prompt.md` | JSON格式，含代码参数 | GPT-4/Claude 3/Kimi |
| 快速风格判断 | `quick-analysis-prompt.md` | YAML格式，快速概览 | 任意模型 |
| 批量帧分析 | 见下方批量提示词 | 对比表格+统一配置 | GPT-4/Claude 3 |

## 批量分析提示词（多帧对比）

```markdown
## 角色
你是MV视觉分析专家。我将提供3张视频帧（开头/中间/结尾），请分析风格一致性。

## 分析步骤

### 步骤1：逐帧分析
对每张图片，提取：
1. 主色调（颜色名称+HEX）
2. 背景模糊程度（无/轻/中/重）
3. 字幕位置+颜色+特效
4. 特殊视觉效果

### 步骤2：一致性检查
- 三张帧风格是否统一？
- 色调是否一致？
- 字幕样式是否有变化？
- 是否有明显的场景切换？

### 步骤3：提取核心配置
基于分析，给出统一的生成参数：
- 推荐色调代码
- 背景处理参数
- 字幕样式配置
- 动画效果建议

## 输出格式

```json
{
  "frame_analysis": [

```

## 提高成功率的技巧

### 1. 图片预处理
- 确保图片清晰度足够（建议宽度不小于800px）
- 如果是视频截图，选择有代表性的帧
- 避免过度压缩的图片

### 2. 提示词增强
- **明确格式要求**："请以JSON格式输出"
- **提供示例**："参考以下示例输出..."
- **约束范围**："从以下选项中选择..."
- **技术关联**："对应PHP GD函数是..."

### 3. 追问策略
如果首次分析不够详细，使用追问：

```
基于刚才的分析，请补充：
1. 具体的颜色HEX代码（如#C00000）
2. 模糊程度的像素值估计
3. 字体大小的相对比例
4. 对应的PHP GD代码示例
```

### 4. 颜色提取技巧
```
请提取图片中的主要颜色：
1. 背景主色（画面占比最大的颜色）
2. 文字颜色
3. 特效颜色（如阴影、发光）
4. 遮罩层颜色

请以HEX格式输出所有颜色代码。
```

## 常见问题

### Q: 模型无法识别字幕颜色
**解决方案**：
```
请仔细观察图片中的文字部分：
1. 文字是什么颜色？（如红色、白色、黄色）
2. 文字是否有描边/阴影？描边是什么颜色？
3. 请提供文字的HEX颜色代码
```

### Q: 模型对模糊程度描述模糊
**解决方案**：
```
请判断背景模糊程度：
- 轻度模糊：还能看清背景细节，如人物轮廓
- 中度模糊：只能辨认大致形状
- 重度模糊：几乎完全看不清，只剩色块

请选择其中一项，并估计模糊半径（1-20px）。
```

### Q: 模型输出格式不一致
**解决方案**：
强制使用模板：
```
请严格按照以下模板填空：

风格类型：____（从"复古/现代/梦幻/暗黑/清新"选择）
主色调：____（颜色名+HEX，如"暖橙色#FF8C42"）
背景处理：____模糊 + ____遮罩
字幕位置：____（上/中/下）
字幕颜色：____（HEX代码）
字幕特效：____（阴影/发光/描边/无）
字体大小：____（占画面百分比）
```

## 实际使用示例

### 单帧快速分析
```bash
# 将以下提示词+图片发送给AI
"请分析这张MV截图的视觉风格。重点关注：
1. 整体色调（暖/冷/复古）
2. 背景处理方式
3. 字幕位置和样式
4. 任何特殊视觉效果

请以JSON格式输出，包含颜色HEX代码。"
```

### 批量分析工作流
```bash
# 提取视频多帧
ffmpeg -i input.mp4 -vf "fps=1/30,scale=640:-1" frames/frame_%03d.jpg

# 选择代表性的3帧：开头、中间、结尾
# 发送给AI进行批量分析
```

### 自动化建议
可以将提示词模板化，通过脚本调用API：

```php
$prompt = file_get_contents('prompts/quick-analysis-prompt.md');
$imageBase64 = base64_encode(file_get_contents('frame.jpg'));

// 调用AI API
$response = callAIAPI([
    'prompt' => $prompt,
    'image' => $imageBase64,
    'model' => 'gpt-4-vision'
]);

// 解析结果
$config = parseAIResponse($response);
generateMV($config);
```

## 推荐工作流

1. **提取帧**：使用FFmpeg提取关键帧
   ```bash
   ffmpeg -i video.mp4 -ss 00:00:01 -vframes 1 frame1.jpg
   ffmpeg -i video.mp4 -ss 00:01:30 -vframes 1 frame2.jpg
   ffmpeg -i video.mp4 -ss 00:03:00 -vframes 1 frame3.jpg
   ```

2. **批量分析**：使用"批量分析提示词"分析3帧

3. **提取配置**：从AI输出中提取颜色、字体等参数

4. **生成MV**：使用提取的配置运行PHP脚本

5. **微调**：根据生成效果，调整参数重新生成
